# 🏠 HHA

**AI 기반 집 분위기 분석 및 가구 추천 시스템**  
사용자의 집 사진을 분석하여 인테리어 분위기를 진단하고, 분위기에 맞는 가구를 추천해주는 스마트 홈 어시스턴트 앱입니다.

---

## 주요 기능
- 집 내부 이미지 업로드 및 AI 분위기 분석
- 분석 결과 기반 가구 스타일 추천
- 결과 저장 및 공유 기능
- 유저별 선호 스타일 분석 및 맞춤형 제안
